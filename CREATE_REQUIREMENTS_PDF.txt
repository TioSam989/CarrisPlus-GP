========================================
HOW TO CREATE REQUIREMENTS PDF
========================================

OPTION 1 - Terminal (Easiest):
------------------------------
Run this command:

    python create_requirements_pdf.py

Or:

    python terminal_pdf.py


OPTION 2 - Manual (Always Works):
----------------------------------
1. Run: python create_requirements_pdf.py
   (This creates CarrisPlus_Requirements.html)

2. Open CarrisPlus_Requirements.html in your browser

3. Press Ctrl+P

4. Select "Save as PDF"

5. Click Save
